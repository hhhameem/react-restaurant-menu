## `If you want to run locally`

**Run Those command in order.**

- ```sh
   git clone https://github.com/hhhameem/react-restaurant-menu.git
  ```

- ```sh
    cd react-restaurant-menu
  ```
- ```sh
    npm install
  ```
- ```sh
    npm start
  ```
